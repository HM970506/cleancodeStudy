### 🐾 RULE
1. 정해진 분량(약 50p)을 각자 스터디 시간까지 공부하고 github에 정리하기  
2. 랜덤으로 한 명을 뽑아서 자신이 공부한 내용을 발표하기  
3. 공부한 내용에 대해 토의하기  
4. 가장 내용 정리를 잘 해온 스터디원을 <b>Best Reader🏆</b>로 선정!
