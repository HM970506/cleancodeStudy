### 🐾 RULE
1. 정해진 분량을 각자 스터디 시간까지 공부하고 정리해오기

3. 공부한 내용에 대해 토의하기
4. 가장 잘 정리해온 사람을 Best Reader로 선정하고! 
5. git에 각자 정리해온 내용을 통합해서 올리기
